## Run Locally

Clone the project

```bash
  git clone https://github.com/Yeabtsega1/Shopping site
```

Go to the project directory

```bash
  cd Shopping site
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
